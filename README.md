# Swap and Play: A Motion Activated Smart Bluetooth Speaker

## Overview

Swap and Play is a Student Innovative Project (SIP) that introduces a smart Bluetooth speaker featuring gesture-based control and a replaceable rechargeable battery system.

The project aims to improve user convenience, reduce electronic waste, and provide a more interactive audio experience through motion-based controls.
